# Documentacion Profesional - FinancePro

Esta carpeta contiene una explicacion completa, profesional y extensa del funcionamiento de la aplicacion, con enfoque tecnico y funcional.

Archivos incluidos:

- `01_base_de_datos_completa.md`: diseno completo de la base de datos, relaciones, decisiones y mapeo por modulo.
- `02_funcionamiento_general_de_la_app.md`: explicacion integral de como funciona la plataforma de extremo a extremo.
- `03_explicacion_detallada_database_setup.md`: explicacion detallada del script `database_setup.py`, bloque por bloque.

Objetivo:

- Tener una base formal para entrega academica o profesional.
- Facilitar mantenimiento futuro y escalado de la aplicacion.
- Explicar no solo "que hace", sino "por que se hizo asi".
